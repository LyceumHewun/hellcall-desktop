# Suggested commands
- `bun install` — install frontend dependencies.
- `bun tauri dev` — run the desktop app in development.
- `bun tauri build` — produce a production build.
- `cargo fmt` (run in `src-tauri/`) — format Rust backend code.
- `cargo check` (run in `src-tauri/`) — compile-check Rust backend.
- `rg -n "pattern" src src-tauri/src` — fast project search on Windows.
- `Get-Content <path>` and `Get-ChildItem` — useful PowerShell file inspection commands in this repo.
